Bienvenidos al proyecto Foodies

El proyecto de referencia en la venta de comida a domicilio.

¿Qué es Foodies?

  • Foodies es una aplicación con la que podrás pedir comida a un restaurante y recibirla sin tener que salir de casa.
  • Esta aplicación web es además un proyecto para el ciclo superior de Desarrollo de Aplicaciones Web (DAW)

¿Quién impulsa este proyecto?

  • El autor de la aplicación base es Ismael Garcia Lorenzo (https://github.com/Isgarlo87); Alumno de segundo del Ciclo Formativo de Grado Superior denominado Desarrollo de Aplicaciones Web.
  • La aplicación ha sido desarrollada mediante el uso de los conocimientos y programas hechos en clase.
  • El programa esta creado a partir de las indicaciones de un cliente concreto para el futuro uso de la aplicación.

¿Qué se pretende con este proyecto?

  • Realizar una aplicación tipo usable para el futuro.
  • Participar en la creación de una aplicación web y mejorarla con el paso del tiempo.
